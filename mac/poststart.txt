Customize VS Code
- Install the 'code' shell command to Terminal 
    -  Open VS Code, click ⇧ + ⌘ + P (shift + command + p)
    -  Type 'shell command' and hit Return
- Open Settings by pressing ⌘ + , (command + ,)
    - Set font size to: 28
    - Set tab size to:  4
    - Set font family to: Menlo, Monaco, 'Courier New', monospace
    - Set tab size to: 4
    - Set word wrap to: wordWrapColumn 
    - Set word wrap column to: 80
    - Set auto indent to: full
- Install the following extensions
    - Auto Rename Tag
    - Bracket Pair Colorizer
    - ES7 React/Redux/GraphQL/React-Native snippets
    - Indenticator 
    - Prettier 
    - Visual Studio IntelliCode 
    - C/C++
    - Debugger for Java
    - Java Dependency Viewer 
    - Java Extension Pack 
    - Java Test Runner 
    - Language Support for Java(TM) by Red Hat
    - Maven for Java
